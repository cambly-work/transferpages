# Morrison Premium Transfer — static website

Сайт премиальных частных трансферов по Бразилии и Южной Америке (HTML/CSS/JS, без сборки).

## Структура проекта

- `index.html` — главная: оффер, преимущества, ключевые маршруты, ссылки на локальные страницы.
- `brazil-transfers.html` — междугородние маршруты по Бразилии и ориентиры по стоимости.
- `airport-transfer.html` — трансферы в/из аэропортов (FLN, POA) и связка с межгородом.
- `international.html` — международные маршруты (Бразилия → Уругвай / Парагвай).
- `partners.html` — отдельная страница для партнеров (риэлторы, юристы, отели, relocation).
- `contact.html` — контакты и сценарий бронирования.
- `balneario-camboriu.html`, `porto-alegre.html`, `florianopolis.html` — локальные SEO-страницы.
- `assets/style.css` — единый стиль сайта.
- `assets/script.js` — мобильное меню, активный пункт навигации, текущий год в footer.
- `assets/images/hero-car.svg` — ключевая визуализация hero.

```bash

```

## Как менять цены

Основные ценовые ориентиры находятся в:
- `index.html` (блок с условиями),
- `brazil-transfers.html` (таблица маршрутов),
- локальные страницы при необходимости (примерные упоминания).

После изменения цен проверьте, что формулировки на всех страницах остаются согласованными.

## Как редактировать ключевые CTA

Главные CTA находятся в трех точках каждой ключевой страницы:
1. в hero (первый экран),
2. в середине контента (карточки / блоки),
3. в нижнем `cta-band`.

Проверьте после правок:
- ведут ли кнопки в правильные каналы (`WhatsApp`, `Telegram`, `contact.html`),
- нет ли дублирующих или конфликтующих формулировок.

## Локальный просмотр

Откройте `index.html` напрямую в браузере или используйте любой простой static server.

Пример:

```bash
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`.
