Pix, cartão, boleto: o que é mais prático no Brasil

Como recebemos pagamentos, o que é mais rápido para viagens pontuais, o que serve melhor para empresas recorrentes.

Pix é a transferência instantânea brasileira — 24/7 e sem taxa. Para transfers pontuais é o mais rápido. Aceitamos cartão também, mas via adquirente — com taxa adicional. Para parceiros recorrentes, boleto com pagamento posterior, NF-e no CNPJ, prazo de 7 a 14 dias.

Esta é uma versão piloto do artigo. O conteúdo completo está em preparação. Se tem uma dúvida específica sobre o tema — escreva no WhatsApp, respondemos em uma hora.
