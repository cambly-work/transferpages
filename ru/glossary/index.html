Глоссарий

Бразильские термины,
с которыми вы столкнётесь.

NF-e, CNPJ, Pix, DPVAT, CNH, LGPD — каждый термин в одной фразе.

NF-e
Nota Fiscal Eletrônica — обязательная электронная фактура в Бразилии. Выставляется на CNPJ компании-клиента. Для бухгалтерии — основной отчётный документ по расходам на трансфер.
CNPJ
Cadastro Nacional da Pessoa Jurídica — налоговый номер юр. лица в Бразилии. Аналог ИНН для компаний. Без CNPJ нельзя получить NF-e.
CPF
Cadastro de Pessoas Físicas — налоговый номер физ. лица. Аналог ИНН для людей. Нужен для безналичных платежей крупного размера.
Pix
Мгновенная межбанковская оплата в Бразилии. Работает 24/7, без комиссии для частных лиц. Самый быстрый способ оплаты разового трансфера.
DPVAT
Обязательная страховка пассажиров наземного транспорта в Бразилии. Покрывает базовые случаи травм или гибели. Действует автоматически.
CNH
Carteira Nacional de Habilitação — водительские права в Бразилии. Для пассажирских перевозок нужна категория D или B + EAR (Exercício de Atividade Remunerada).
BR-101
Прибрежная федеральная трасса вдоль восточного побережья Бразилии. Основной маршрут на юге: SP → Curitiba → Florianópolis → Porto Alegre.
BR-116
Самая длинная федеральная трасса Бразилии, проходит от Fortaleza до Jaguarão (граница с Уругваем). Основной деловой коридор.
LGPD
Lei Geral de Proteção de Dados (Law 13.709/2018) — бразильский аналог GDPR. Регулирует обработку персональных данных. Контролирует ANPD.
ANPD
Autoridade Nacional de Proteção de Dados — национальный регулятор персональных данных. Принимает жалобы пассажиров, если права по LGPD нарушены.
CDC
Código de Defesa do Consumidor — основной потребительский закон Бразилии. Регулирует претензии, отмены и компенсации в B2C.
Mercosur / Mercosul
Экономический блок: Бразилия, Аргентина, Уругвай, Парагвай. Между странами действует упрощённый таможенный режим для пассажиров.
Chuí
Сухопутный пограничный город между Бразилией и Уругваем. Основной автомобильный переход с юга Бразилии в Mercosur.
